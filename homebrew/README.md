# Boop Homebrew Tap

This is the official Homebrew tap for Boop, a Todoist CLI with purpose.

## Installation

```bash
# Add the tap
brew tap kshehadeh/boop

# Install boop
brew install boop
```

## Updating

```bash
brew update
brew upgrade boop
```

## Uninstalling

```bash
brew uninstall boop
brew untap kshehadeh/boop
```
