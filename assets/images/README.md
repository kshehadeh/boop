# Image Assets

This directory should contain the following image files for the Boop documentation site:

## Favicons
- favicon.ico
- favicon-16x16.png
- favicon-32x32.png
- apple-touch-icon.png
- android-chrome-192x192.png
- android-chrome-512x512.png

## Social Media
- boop-social-card.png (1200x630px recommended for social sharing)

## Logo
- boop-logo.png (for site header)

You can generate favicon files using tools like [favicon.io](https://favicon.io/) or [RealFaviconGenerator](https://realfavicongenerator.net/).
